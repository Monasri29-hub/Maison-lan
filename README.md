# Maison Élan — Premium Restaurant Website Experience 🍽️✨

![Maison Élan Banner](./assets/banner.png)

A luxury European-inspired restaurant and patisserie website designed to showcase a premium dining experience through immersive storytelling, elegant UI, and modern web interactions.

Maison Élan is a fictional premium cafe and restaurant brand created as a frontend portfolio project to demonstrate how hospitality brands can transform their online presence into a sophisticated digital experience.

The goal of this project was not just to build a restaurant website, but to create a **digital brand experience** that communicates ambience, craftsmanship, and emotion before a customer even visits.

---

## 🌟 Project Vision

Most restaurants today rely heavily on Instagram, Google Maps, and food delivery platforms. However, a premium brand needs a dedicated digital space where customers can:

- Understand the brand story
- Explore the menu
- Experience the ambience
- View signature dishes
- Build trust through reviews
- Easily reserve a table

Maison Élan solves this by creating a visually immersive website inspired by luxury hospitality, boutique hotels, and premium lifestyle brands.

# 📱 WhatsApp Reservation System

Instead of building unnecessary payment infrastructure, Maison Élan uses a simple reservation workflow.

Features:

- Reserve Table button
- WhatsApp integration
- Pre-filled reservation message
- Customer enquiry flow

Reservation message format:

```

Hello Maison Élan, I would like to reserve a table.

Name:
Date:
Time:
Number of Guests:
Special Requests:

```

Customers complete the details and send the message directly through WhatsApp.

Maison Élan focuses on premium dining experience rather than delivery.

## Why WhatsApp reservations?

For many restaurants:

- Customers prefer direct communication
- Owners can personally confirm availability
- No unnecessary booking system complexity
